Mario Jump

This is a game project purely created for fun and with the goal of create some cool animation using the time function in Javascript.

LINK TO WEBSITE

Case & Develop

This is a feature that I have implemented in another project and I thought that one day I will like to expand with more option to create a real playable game.

I grow up with Super Mario and it's a character that anyone are well aware of, so I have used the classic 2D game to create an animation pure javascript based.

Mario, the background and the burger box are pictures that they are triggered by events using a timer function in Javascript. It is cool and essentially it is the base to develop a nice animation concept that I will carry on in my future projects.

Development time

The project was accomplished in a period of 1 week.
