Case & Develop

Version 2.0 of a feature that I had created only with the setTime function in Javascript, now it is a real playable game.

I grow up with Super Mario and it's a character that anyone are well aware of, so I have used the classic 2D game to create an animation pure javascript based.

I was able to create to game using the JS library Kaboom.js that gave me the possibility to implement differents aspects of the game using their in-built funtions
