LINK TO WEBSITE

Case & Develop

First project where we had the freedom to develop our own desing and structure. The task was to develop a new and fresh concepts like logo, posters and a website for an organization no-profit that has as a purpose to raise awareness to its users about the lack of green inside the hearth of the metropolis around the world.

The logo is a mix between different partnership that this no-profit organization has. It takes inspiration from Greenpeace and Save the Tree.

The website is entired build with HTML and CSS without using any framework or libraries. It is fully responsive and it has 9 subpages that are fully implemented and functional.

Development time

The project was accomplished in a period of 3 weeks.
